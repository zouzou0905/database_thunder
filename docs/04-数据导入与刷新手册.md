# 04-数据导入与刷新手册

## 启动虚拟环境

```powershell
cd F:\database
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
.\.venv\Scripts\Activate.ps1
```

## 检查 CSV

导入前先检查文件结构：

```powershell
python scripts\inspect_csv.py --input "C:\Users\admin\Desktop\aba数据\2026-04" --output data\reports\2026-04-inspect.json
```

## 导入月份数据

```powershell
python scripts\import_csv.py --month 2026-04 --marketplace UK --input "C:\Users\admin\Desktop\aba数据\2026-04"
```

如果某个月后续还有补充文件，可以继续用同一条命令导入同月文件夹。系统会通过文件 hash 避免重复导入。

需要强制重新处理时才使用：

```powershell
python scripts\import_csv.py --month 2026-04 --marketplace UK --input "C:\Users\admin\Desktop\aba数据\2026-04" --force
```

## 计算趋势和缓存

导入后运行：

```powershell
python scripts\calculate_trends.py --analysis-month 2026-04 --marketplace UK
```

这个命令默认刷新：

```text
keyword_monthly_trends
keyword_ops_monthly
keyword_selection_candidates_monthly
keyword_compare_snapshot_monthly
```

如果数据不是按月份顺序导入，或补导了早期月份，直接重算全部已导入月份：

```powershell
python scripts\calculate_trends.py --all --marketplace UK
```

指定范围：

```powershell
python scripts\calculate_trends.py --all --marketplace UK --from-month 2025-04 --to-month 2026-04
```

## 只刷新机会池缓存

页面提示“实时计算，查询较慢”时，通常说明当前月份缓存缺失。可单独补：

```powershell
python scripts\refresh_product_selection_cache.py --analysis-month 2026-04 --marketplace UK
```

## 日常导入模板

```powershell
cd F:\database
.\.venv\Scripts\Activate.ps1

python scripts\inspect_csv.py --input "C:\Users\admin\Desktop\aba数据\YYYY-MM" --output data\reports\YYYY-MM-inspect.json

python scripts\import_csv.py --month YYYY-MM --marketplace UK --input "C:\Users\admin\Desktop\aba数据\YYYY-MM"

python scripts\calculate_trends.py --analysis-month YYYY-MM --marketplace UK
```

## 卡住时怎么判断

导入几百个文件时可能看起来像假死。当前导入脚本应定期打印进度。

如果长时间没有输出，先不要强行关闭，另开一个 PowerShell 检查数据库或进程状态。只有确认进程无响应时再重启任务。
