# 04-数据导入与刷新手册

本文是日常导入和刷新命令速查。完整导入流程见 [08-数据导入完整流程.md](08-数据导入完整流程.md)。

## 基础环境

所有命令默认从项目根目录执行：

```powershell
cd D:\database\database_thunder-main
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
.\.venv\Scripts\Activate.ps1
```

涉及 `app.*` 导入的 Python 命令需要：

```powershell
$env:PYTHONPATH = "backend"
```

管理后台相关命令需要：

```powershell
$env:PYTHONPATH = "backend;admin"
```

`backend` 必须在 `admin` 前面，避免 `app` 包名冲突。

## 检查 CSV

导入前先检查文件结构：

```powershell
python scripts\inspect_csv.py `
  --input "D:\PostgreSQL\data\aba数据\YYYY-MM" `
  --output data\reports\YYYY-MM-inspect.json
```

## 导入月度数据

```powershell
python scripts\import_csv.py `
  --month YYYY-MM `
  --marketplace UK `
  --input "D:\PostgreSQL\data\aba数据\YYYY-MM"
```

同一批文件会通过 SHA256 文件 hash 去重，重复执行不会重复导入同一文件。

如确认需要强制重导：

```powershell
python scripts\import_csv.py `
  --month YYYY-MM `
  --marketplace UK `
  --input "D:\PostgreSQL\data\aba数据\YYYY-MM" `
  --force
```

## 刷新趋势、机会池和横向对比缓存

导入后通常执行完整刷新：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\calculate_trends.py --all --marketplace UK
```

该命令会执行：

```text
keyword_monthly_trends
keyword_ops_monthly
keyword_selection_candidates_monthly
keyword_compare_snapshot
keyword_compare_range_cache 常用区间预热
```

如果只处理单个分析月：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\calculate_trends.py --analysis-month YYYY-MM --marketplace UK
```

如果只想刷新趋势和运营评分，不刷新机会池缓存：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\calculate_trends.py --all --marketplace UK --skip-cache-refresh
```

## 独立刷新脚本

机会池缓存：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\refresh_product_selection_cache.py --analysis-month YYYY-MM --marketplace UK
```

横向对比全量快照：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\refresh_compare_snapshot_only.py --marketplace UK
```

横向对比常用区间预热：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\prewarm_compare_range_cache.py --marketplace UK
```

固定区间快照：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\refresh_compare_intervals.py --marketplace UK
```

场景快照：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\refresh_compare_scenarios.py --marketplace UK
```

## 统计信息维护

导入大量数据或重建缓存后，可执行：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\vacuum_analyze.py
```

验证核心表统计：

```powershell
$env:PYTHONPATH = "backend"
.\.venv\Scripts\python.exe scripts\verify_core_table_stats.py
```

## 常见判断

```text
机会池很慢：检查 keyword_selection_candidates_monthly 是否已刷新
横向对比全量区间很慢：检查 keyword_compare_snapshot 是否存在
横向对比自定义区间很慢：检查 keyword_compare_range_cache 是否预热
节日筛选没有结果：检查 keyword_holiday_tags 是否已刷新
PostgreSQL 重启后横向对比变慢：keyword_compare_range_cache 是 UNLOGGED，需要重新预热
```
